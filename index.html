⬇️ Descargar script para la laptop

Este script se ejecutará en segundo plano y actualizará automáticamente el archivo hosts según lo que configures aquí.

📝 Instrucciones:
  1. Descarga el script y colócalo en una carpeta de la laptop.
  2. Descarga la clave de servicio (Service Account JSON) desde Firebase Console → Configuración → Cuentas de servicio.
  3. Renombra el archivo de clave como serviceAccountKey.json y ponlo en la misma carpeta que el script.
  4. Ejecuta el script como administrador (clic derecho → Ejecutar como administrador) para que pueda modificar el archivo hosts.
  5. ¡Listo! Se actualizará cada 10 segundos.
